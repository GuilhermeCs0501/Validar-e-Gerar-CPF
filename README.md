Validador e Gerador de CPF JS  
Projeto de um sistema para validar e gerar CPFs brasileiros usando HTML, CSS e JavaScript.

Funcionalidades  
Validação de CPF  
Geração de CPF válido  
Verificação dos dígitos verificadores  

Como usar  
Digite um CPF para validar ou clique no botão para gerar um CPF válido automaticamente.
